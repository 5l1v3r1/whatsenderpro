FAQ – Issues, errors and info

FAQ WHATSENDER BULK WHATSAPP SENDER

1. Why Google Chrome gime me error “Incorrect phone number” after scan QR-code?
For incorrect phone number please wait it will skip it or ensure that the first number on your bulk is a right number

2. What can I do to insert “Full Name” in Russian language, cause now when sending users receive “? ? ???” instead of a Name?
When you import number with name in russian ensure that your file is unicode not ANSI

3. It’s sending each line as a separate message.. How can I send eentire message as one message with Line breaks?
From Advanced settings try to set “text inject” sending type

4. I click Send button, it opens Whatsapp Web but it doesn’t start sending messages, why?
Please be sure you have last version of software, go in DOWNLOAD page to download last version

4bis. I don’t see the QR code in Google Chrome
Try to:
-delete cookies in Chrome
-disable your proxy and antivirus
-In Chrome go in Settings > Advanced and disable Hardware acceleration

5. I have problems to import csv file with names and numbers, how can I solve it?
Be sure to import csv with the “;” separator, with names in 1st field and numbers in the 2nd field.

6. I receive Error “Bulk has been Stopped”… how fix it?

The cause of the problem is that the app cannot connect to chrome so please do the following:
1) Re-boot the PC
2) Right click on the icon app and select “Run as Administrator”
3) Set Chrome as Default Browser
4) Disble any adblocker in Chrome (if you have it)
5) If the problem still exist, uninstall and reinstall Chrome
6) From setting select text inject

7. I receive Error “Invalid number or blocked Number”… how fix it?

  1. Please restart Chrome and restart the program

8. It gives error after installing: Problem Event Name: CLR20r3

You have to install the .NET Framework 4.6 in your PC:  https://dotnet.microsoft.com/download/thank-you/net46

9. The customization /personalization of message doesn’t work, why?

Try to put in the message the tag [[fullname]]. Also you have to disable the “turbo mode” option.

10. I got the error “Phone number shared via url is invalid”

It means that the recipient’s number has no Whatsapp or it is inexistent or it has not the Country prefix ( +91 for India, +1 for USA..)

11. My Whatsapp number got banned, why this happened? how to avoid this?

IMPORTANT NOTE: WhatSender allows you to easy communicate with recipients that expect receive your messages (Eg. clients , students , patients , etc..), and it’s NOT A SPAM TOOL. We are not responsible if you got banned.

Below some tips to reduce the risk of being banned:
1. We suggest to not use your main personal account  (you can use a new account number)
2- Just try to use your new account for few days sending and receiving and after sending bulk from it
3. Send account number (that you want use it to send bulk) as vcard from your personal account with friends and family ask them to save it and to message you on it
4- When you start send bulk from new account start with small bulks
5- Try to write message in a way that recipient do not report it as spam (this is the most critical issue with new accounts)
6- Try to use Multi Message and/or Spintax options on the application, which allow you to write same message in different way
7- Use anti block module application in setting (add your numbers and your friends number that you usually chat with them and set initiate dialog with them after x amount of messages)
8. Give your account more credibility by joining groups
9. Do not send more than 600 messages per hour. WhatSender has a unique capping option where you can tell WhatSender when to stop sending messages.
10. Send messages only to opt-in users
12. What is the difference between the FREE and the PRO versions?
In the FREE version you can add your contacts individually (mass importing from csv is disabled). In the FREE version also the features of “filtering whatsapp numbers” and “multi-accounts” are disabled.
In the PRO version you can import your contacts numbers from csv files and also by copying and pasting. In the PRO version you can also use the filtering options and add multiple accounts.
